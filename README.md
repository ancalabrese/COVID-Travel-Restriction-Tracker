# COVID-Travel-Restriction-Tracker
A tracker for travel restrictions updates (UK only for now)

If you want to contribute with travel restrictions for your own country please submit a pull request. 
Data format: <br>
  `"country_name":{
    "update_date":{
      "list1":{
        "country_name":"notes",
        [...]
      },
      [...]
    }
  }`
